BigO 표기법 특징:
================


1. 상수항 무시
ex O(2N) ~= O(N)

2. 최고차항 빼고 무시
ex O(N^2 + N + 1) ~= O(N^2)


3. 상수함수 < 로그함수 < 선형함수 < 다항함수 < 지수함수
순으로 시간 지연이 발생한다.




대표적인 Big O 표기법
==================

1. O(1) : 스택에서의 push.pop
2. O(logN) : 이진트리
3. O(N) : for문
4. O(NlogN) : 퀵 정렬, 병합 정렬, 힙 정렬
5. O(N^2) : 이중 for문, 삽입 정렬, 버블 정렬, 선택 정렬
6. O(2(N^2)) : 피보나치 수열
